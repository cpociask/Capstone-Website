Description

Our client recieves terabytes of astronomical data each day. Visualizing this data is currently very poor, or even broken. We are building a visualization tool that will make visualizing this data easy and attractive, while being performant and hosted at NAU.

High Level Requirements
  • Hosted on NAU's linux servers
  • Using a Python or R based visualization library
  • Should be interactive
  • Performant for potientally millions of data points

Technologies Used

  • Git/GitHub VC and repository host
  • Will be hosted on Linux servers at NAU
  • Currently considering Dash, Streamlit, Shiny
